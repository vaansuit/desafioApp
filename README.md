# desafio_app

Aplicativo em Flutter utilizando SQFLite como banco de dados. 
Funções: 
  Cadastrasr cliente.
  Cadastrar produtos.
  Consulta de vendas.
  Fake button sincronizar.

Como rodar:

Baixe o projeto ou faça um clone.
Rode o comando flutter pub get para baixar todas as dependências.
